Color Setup for FAR 3
-------------------------

Плагин для настройки цветов FAR3. Позволяет сохранять/восстанавливать цветовые 
палитры в файлах в собственном формате *.farpal.
Поддерживает TrueColor палитры.

Автор: Максим Русов (Max.Rusov@gmail.com)

Последние версии и исходные тексты:
http://code.google.com/p/far-plugins

-------------------------------------------------------------------------------
Использование Color Setup  через командную строку:

 pal:/Команда=Параметр 

Команды:

  /Save=ИмяФайла
  /Load=ИмяФайла

  Сохранение/считывание палитры FAR в файл формата *.farpal
  Удобно настроить ассоциацию:

  *.farpal: pal:/load="!\!.!"
