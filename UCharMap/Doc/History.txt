
Ver 1.0.1

 + В нижней строке таблицы символов может отображаться имя текущего символа.
   Для этого в каталоге плагина должен находится файл "NameList.txt"
   Сам файл в архив не включен и качается отсюда (размер ~1M):

   http://unicode.org/Public/UNIDATA/NamesList.txt

   Если файл NamesList.txt отсутствует, в нижней строке отображается имя
   диапазона символов, из файла GroupList.txt

 + Добавлена команда перехода на определенный диавазон символов: Ctrl-G,
   или клик на нижней строке таблицы. В списке диапазонов работает 
   фильтрация при наборе.

Ver 1.0.2

  + Добавлена возможность выбора шрифта (команда Ctrl-F или клик в верхней
    строке таблицы на имени шрифта). CharMap анализирует диапазоны символов
    представленные в шрифте, отсутствующие символы помечаются цветом.

    Команда Ctrl-H позволяет скрыть лишние строки таблицы (такие строки,
    в которых нет ни одного символа из текущего шрифта).

  + Интеграция с FarHints (требуется FarHints ver 1.0.14). При включенной
    поддержке хинтов в диалогах, показывается подсказка с информацией о
    символе под курсором и его начертание в текущем шрифте.

  + Окно CharMap'а можно растянуть по высоте с помощью команды Ctrl-M или
    кликом в иконку максимизации.

  + Некоторые опции CharMap'а можно настроить через файл Settings.reg

  * Исправлено несколько ошибок

Ver 1.0.3
  
  * При копировании символа в поле ввода (Ins) указатель автоматически 
    перемещается на следующий символ

  * Если в поле ввода скопировано несколько символов, то по Enter они
    все вставляются в текущий контекст ввода.

  * Коды символов отображаются полностью (4-мя байтами)

  * Для работы плагина требуется FAR/2 Build 995+
